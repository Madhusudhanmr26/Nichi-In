What is HTML?

HTML stands for Hyper Text Markup Language. It is a Text based language for developing Web pages. The Web Pages are executed inside a software called Browser. Browser is a thin software used to view or execute HTML and HTML based Pages.
Its purpose is to create web pages in a predefine structures.

HTML is a markup language. It allows the computers to communicate with each other and present the text in a processed manner. This way of processing the text is called as Markup. Tags are used to define the markups of the Text.

HTML Breakup

HyperText is a text that has a link within it, on which when clicked would bring a new content probably from another page or another part of the same page. Apart from Text, HyperText might include Tables, Lists, Forms, Images etc.

Markup Language: Is one that uses Tags to define elements within a Document. The Tags are easy to understand, they contain human readable words like table, h1, p, div, link, title and so forth. Every tag will have a sp meaning of its own and the HTML Parser will perfrm a perticular operation when it comes across that tag.

Features of HTML

  1. HTML describes a structure of the Web Page.
  2. Elements are represented as Tags that are human readable
  3. They are static in nature. The browser should reload the page when it wants to see the fresh content
  4. Its very simple and easy to use. It does not contain any logical code like other programming languages
