Lists in HTML

Lists are of 3 types in HTML 5. They are:

  • ordered List
  • UnOrdered List
  • Description List

    • Ordered List

    When U wish to list a group of items in a definite order like numberings, charecters, they they are created using a tag <ol>. Every List Item is placed under the tag <li> Both li and ol are paired Tags. U could use an attribute type to define how the numbering should happen. It could be any of the following values:

    1. 1
    2. I
    3. i
    4. a
    5. A

    UnOrdered List

    When U wish to list a group of items in an indefinite order without any numberings, charecters, they they are created using a tag <ul>. Instead of numberings, U will have bullets placed to identify the list item. Every List Item is placed under the tag <li> Rest of the features are similart to Ordered List. U could use an attribute type to define the shape of the bullet. It could be any of the following values:

    • disc
    • circle
    • square
    • none

    Description Lists

    HTML has a new style called Description List or Simply Definition List that will hold entries listed like a Dictionary. The Definition List is good way to present a list of tems and its terminologies. The Definition will be provided by a tag called <dt> and the details of it will be placed under <dd> Tag.
