HTML Tags

Every Tag provides a sp meaning to the browser while it processes the page. The processing of the page is called as PARSING in HTML. Each tag performs its own function and helps in creating a proper structure of the HTML Document. However U cannot create UR own tags in HTML. They are all predefined.

Types of Tags

  • Paired Tags: They will have a start tag followed by an End Tag
  • Unpaired Tags: They are single tags which will have an end Tag within them

    HTML Paragraph Tag

    The <p> Tag is used to define a paragraph within the HTML Document. It provides basically a paragraph like finishing, it will start from a seperate line and end the content and a carrier to the next line. Technically the browser adds a blank line before and after the <p > tag.

    The Linking Tag: The A Tag

    HTML Links are defined using <a> Tag(Anchor Tag). It allows to provide links to another file, image or any kind of file that can be accessible from its location. It is provided with an attribute(Additional property) called href which contains the location of the file. This could be a relative path or Absolute path of the file.
    Anchor tags can also point to the different locations of the same page using attribute called id.

    Images

    The Image tag is used to add Images to the HTML Document. The Tag <img> is used to add images in the document. The "src" attribute is used to specify the location of the image to be display. The size of the image can be controlled using width and height attributes.
