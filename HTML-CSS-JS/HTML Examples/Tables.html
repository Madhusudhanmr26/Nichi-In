Tables in HTML

Presenting the data in the form of Rows and Columns is what makes Tables. Used to display tabular data like List of Employees, Customers, Billings etc. Starts with a table Tag, followed by each row represented as tr tag. Each tr will have either a td(TableData) or a th(TableHeader). Every td and tr has to have an ending tag.
Many a time, the presentation of the data into a Table would be dynamic which is done using any scripting language.
