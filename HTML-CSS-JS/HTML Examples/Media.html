Media usage in HTML5

Since HTML5, media files like Audio and Video can now be placed in HTML documents without a need for any external media players lile Flash and Silverlight. These are the new features of HTML5 which is supported only by the latest browsers.
HTML has video tag that can be used to play a video inside a HTML Document...
Currently the Following formats are only supported in HTML5:

Video File Format Media Type
.mp4 video/mp4
.ogg video/ogg
.webM video/webM
